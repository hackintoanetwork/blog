The Problem: Same-Origin Policy (SOP)#

Same-Origin PolicyлҠ” мӣ№ нҺҳмқҙм§Җк°Җ м„ңлЎң лӢӨлҘё м¶ңмІҳ(Origin)м—җм„ң лЎңл“ңлҗң лҰ¬мҶҢмҠӨм—җ м ‘к·јн•ҳлҠ” кІғмқ„ м ңн•ңн•ҳлҠ” ліҙм•Ҳ м •мұ…мһ…лӢҲлӢӨ. л‘җ URLмқҙ к°ҷмқҖ Originмқ„ к°Җм§Җл Өл©ҙ Scheme, Host, Portк°Җ лӘЁл‘җ лҸҷмқјн•ҙм•ј н•©лӢҲлӢӨ.

мҷёл¶Җмқҳ кіөкІ©мһҗ м„ңлІ„(attacker.example.com)м—җм„ң Starlink мһҘл№„(192.168.1.1 лҳҗлҠ” 192.168.100.1)лЎң gRPC мҡ”мІӯмқ„ ліҙлӮҙлҠ” мғҒнҷ©мқ„ мғқк°Ғн•ҙліҙкІ мҠөлӢҲлӢӨ. SOPм—җ мқҳн•ҙ кіөкІ©мһҗлҠ” мҡ”мІӯмқ„ ліҙлӮј мҲҳлҠ” мһҲм§Җл§Ң, Starlink мһҘл№„лЎңл¶Җн„°мқҳ мқ‘лӢөмқ„ мқҪмқ„ мҲҳлҠ” м—ҶмҠөлӢҲлӢӨ.

н•ҳм§Җл§Ң м—¬кё°м„ң лҚ” нҒ° л¬ём ңк°Җ мһҲмҠөлӢҲлӢӨ. Starlinkмқҳ gRPCлҠ” application/grpc-web+protoлқјлҠ” нҠ№м •н•ң Content-Type н—ӨлҚ”лҘј мҡ”кө¬н•©лӢҲлӢӨ. мқҙ н—ӨлҚ”к°Җ м—Ҷмңјл©ҙ gRPC лӘ…л №мқҙ лҸҷмһ‘н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

The gRPC-Web+Proto Header Problem#

мқјл°ҳм ҒмңјлЎң gRPC мҡ”мІӯмқ„ ліҙлӮј л•Ң, мқ‘лӢө к°’м—җ grpc-status:0мқҙ л°ҳнҷҳлҗҳл©ҙ gRPC лӘ…л №мқҙ м •мғҒм ҒмңјлЎң мӢӨн–үлҗң кІғмқ„ мқҳлҜён•©лӢҲлӢӨ. н•ҳм§Җл§Ң grpc-web+proto н—ӨлҚ”к°Җ м—Ҷмқ„ кІҪмҡ° Starlinkм—җм„ң gRPCк°Җ лҸҷмһ‘н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

application/grpc-web+protoлҠ” лёҢлқјмҡ°м Җмқҳ CORS м •мұ…м—җм„ң н—Ҳмҡ©н•ҳлҠ” Content-Type (text/plain, multipart/form-data, application/x-www-form-urlencoded)м—җ нҸ¬н•Ёлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

л”°лқјм„ң JavaScriptлҘј нҶөн•ҙ Cross-Origin мҡ”мІӯ мӢң мқҙ н—ӨлҚ”лҘј м„Өм •н•ҳл©ҙ, лёҢлқјмҡ°м ҖлҠ” CORS Preflight (OPTIONS мҡ”мІӯ)мқ„ лЁјм Җ м „мҶЎн•©лӢҲлӢӨ. Starlink мһҘл№„лҠ” мқҙ Preflight мҡ”мІӯм—җ лҢҖн•ҙ м Ғм Ҳн•ң Access-Control-Allow-Headers мқ‘лӢөмқ„ л°ҳнҷҳн•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ, лёҢлқјмҡ°м Җк°Җ мӢӨм ң gRPC мҡ”мІӯ мһҗмІҙлҘј м°ЁлӢЁн•©лӢҲлӢӨ.

кІ°лЎ м ҒмңјлЎң, SOPлҘј мҡ°нҡҢн•ҳм§Җ м•ҠлҠ” н•ң мҷёл¶Җм—җм„ң Starlinkм—җ gRPC мҡ”мІӯмқ„ ліҙлӮҙлҠ” кІғмқҖ л¶Ҳк°ҖлҠҘн•©лӢҲлӢӨ.

Joshua Smailes мҷё м—°кө¬м§„мқҳ л…јл¬ё “Dishing out DoS: How to Disable and Secure the Starlink User Terminal” (University of Oxford, 2023) м—җм„ңлҸ„ мқҙ м җмқ„ лӢӨмқҢкіј к°ҷмқҙ м–ёкёүн•ҳкі  мһҲмҠөлӢҲлӢӨ.

“However, the POST request used to configure the Starlink dish requires the content-type: application/grpc-web+proto header, making it non-simple. This is the only reason that the Starlink dish is not directly exploitable on modern browsers”

DNS Rebinding : SOP Bypass#

DNS RebindingмқҖ лёҢлқјмҡ°м Җмқҳ SOPлҘј мҡ°нҡҢн•ҳлҠ” кіөкІ© кё°лІ•мһ…лӢҲлӢӨ. мқјл°ҳм ҒмңјлЎң SOP л•Ңл¬ём—җ кіөкІ©мһҗк°Җ л§Ңл“  мӣ№ нҺҳмқҙм§Җм—җм„ң лӮҙл¶Җ л„ӨнҠёмӣҢнҒ¬лЎң HTTP мҡ”мІӯмқ„ ліҙлӮҙкі  мқ‘лӢө к°’мқ„ л°ӣм•„мҳ¬ мҲҳ м—ҶмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң DNS Rebindingмқ„ нҶөн•ҙ мқҙ м ңн•ңмқ„ мҡ°нҡҢн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙ кё°лІ•мқҳ н•өмӢ¬мқҖ, кіөкІ©мһҗмқҳ лҸ„л©”мқёмқҙ к°ҖлҰ¬нӮӨлҠ” IP мЈјмҶҢлҘј лҸҷм ҒмңјлЎң ліҖкІҪн•ҳм—¬, лёҢлқјмҡ°м Җк°Җ лҸҷмқјн•ң OriginмңјлЎң мқёмӢқн•ҳл©ҙм„ңлҸ„ мӢӨм ңлЎңлҠ” лӢӨлҘё нҳёмҠӨнҠёмҷҖ нҶөмӢ н•ҳлҸ„лЎқ л§Ңл“ңлҠ” кІғмһ…лӢҲлӢӨ. Originмқҙ лҸҷмқјн•ҳлҜҖлЎң CORS Preflightмқҙ л°ңмғқн•ҳм§Җ м•Ҡкі , grpc-web+proto н—ӨлҚ”лҸ„ м •мғҒм ҒмңјлЎң м „мҶЎлҗ©лӢҲлӢӨ.

мҰү, DNS RebindingмқҖ SOPмҷҖ CORSмқҳ ліҙнҳёлҘј л¬ҙл Ҙнҷ”мӢңнӮӨкі , Starlink мӢңмҠӨн…ңм—җ лҢҖн•ң gRPC Exploitмқ„ к°ҖлҠҘн•ҳкІҢ л§Ңл“ңлҠ” н•өмӢ¬ кё°мҲ мһ…лӢҲлӢӨ.

кіөкІ©м—җ м•һм„ң м•Ңм•„л‘җм–ҙм•ј н•  л‘җ к°Җм§Җ м •ліҙк°Җ мһҲмҠөлӢҲлӢӨ:

  • attacker.example.com - кіөкІ©мһҗмқҳ лҸ„л©”мқё мЈјмҶҢ
  • 1.2.3.4 - кіөкІ©мһҗмқҳ нҺҳмқҙлЎңл“ң м„ңлІ„ л°Ҹ м•…м„ұ DNS м„ңлІ„ IP мЈјмҶҢ

кіөкІ© кіјм •мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

2лӢЁкі„: м•…м„ұ нҺҳмқҙлЎңл“ң м „лӢ¬

кіөкІ©мһҗмқҳ DNS м„ңлІ„лҠ” attacker.example.comмқҳ мЈјмҶҢлҘј 1.2.3.4лЎң мқ‘лӢөн•©лӢҲлӢӨ. м—¬кё°м„ң мӨ‘мҡ”н•ң м җмқҖ DNSмқҳ TTL(Time To Live)мқ„ 0мҙҲлЎң м„Өм •н•ңлӢӨлҠ” кІғмһ…лӢҲлӢӨ. мқҙлҠ” DNS мәҗмӢңлҘј л№ лҘҙкІҢ л§ҢлЈҢмӢңмјң, н”јн•ҙмһҗмқҳ лёҢлқјмҡ°м Җк°Җ attacker.example.comм—җ лҢҖн•ҙ кі„мҶҚ мғҲлЎңмҡҙ DNS мҝјлҰ¬лҘј н•ҳлҸ„лЎқ л§Ңл“Өкё° мң„н•Ёмһ…лӢҲлӢӨ.

н”јн•ҙмһҗмқҳ лёҢлқјмҡ°м ҖлҠ” 1.2.3.4м—җм„ң м•…м„ұ JavaScriptлҘј лЎңл“ңн•©лӢҲлӢӨ.

4лӢЁкі„: лӮҙл¶Җ л„ӨнҠёмӣҢнҒ¬ м ‘к·ј

мқҙлІҲм—җлҠ” кіөкІ©мһҗмқҳ DNS м„ңлІ„к°Җ 1.2.3.4 лҢҖмӢ  Starlink мһҘл№„мқҳ лӮҙл¶Җ мЈјмҶҢлҘј л°ҳнҷҳн•©лӢҲлӢӨ. Routerмқҳ кІҪмҡ° 192.168.1.1, Dishyмқҳ кІҪмҡ° 192.168.100.1мһ…лӢҲлӢӨ.

кІ°кіјм ҒмңјлЎң н”јн•ҙмһҗмқҳ лёҢлқјмҡ°м ҖлҠ” мһҗмӢ лҸ„ лӘЁлҘҙкІҢ Starlink мһҘл№„мқҳ лӮҙл¶Җ мЈјмҶҢлЎң gRPC мҡ”мІӯмқ„ ліҙлӮҙкІҢ лҗ©лӢҲлӢӨ. лёҢлқјмҡ°м Җ мһ…мһҘм—җм„ң Origin(Scheme + Host + Port)мқҖ лҸҷмқјн•ҳлҜҖлЎң SOPлҘј мң„л°ҳн•ҳм§Җ м•Ҡмңјл©°, CORS PreflightлҸ„ л°ңмғқн•ҳм§Җ м•Ҡм•„ grpc-web+proto н—ӨлҚ”к°Җ м •мғҒм ҒмңјлЎң м „мҶЎлҗ©лӢҲлӢӨ.

gRPC Requests Analysis#

Starlink мһҘл№„лҠ” /SpaceX.API.Device.Device/Handle м—”л“ңнҸ¬мқёнҠёлҘј нҶөн•ҙ gRPC-Web н”„лЎңнҶ мҪңлЎң нҶөмӢ н•©лӢҲлӢӨ.

кҙҖлҰ¬мһҗ нҺҳмқҙм§Җм—җм„ң м ‘к·јн•  мҲҳ м—ҶлҠ” лӢӨмҲҳмқҳ мҲЁкІЁм§„ gRPC лӘ…л №мқҙ мЎҙмһ¬н•ҳл©°, мқҙлҠ” Starlink нҺҢмӣЁм–ҙ лӮҙ rootfs/usr/sbin/м—җ мң„м№ҳн•ң device.proto нҢҢмқјмқ„ м¶”м¶ңн•ҳм—¬ л¶„м„қн•ЁмңјлЎңмҚЁ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

gRPC мҡ”мІӯмқҳ HTTP л°”л””м—җлҠ” ProtobufлЎң мқёмҪ”л”©лҗң hex к°’мқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲмңјл©°, мқҙлҘј л””мҪ”л”©н•ҳл©ҙ к°Ғ лӘ…л №м—җ н•ҙлӢ№н•ҳлҠ” н•„л“ң лІҲнҳёлҘј нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

04. Dishy - Stow / Unstow (м•Ҳн…ҢлӮҳ м ‘кё° / нҺҙкё°)

Starlink Dishy м•Ҳн…ҢлӮҳлҘј л¬јлҰ¬м ҒмңјлЎң м ‘кұ°лӮҳ нҺҙлҠ” лӘ…л №мһ…лӢҲлӢӨ. Stow мӢӨн–ү мӢң мқён„°л„· м—°кІ°мқҙ мҰүмӢң м°ЁлӢЁлҗҳл©°, мӢӨм ңлЎң Dishyк°Җ м ‘нҳ”лӢӨк°Җ нҺјміҗм§ҖлҠ” лӘЁмҠөмқ„ мңЎм•ҲмңјлЎң нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.
